- `female_yuanyang_happy` - 开心/微笑
- `female_yuanyang_sad` - 悲伤/失落

- `female_yuanyang_surprised` - 惊讶/震惊
- `female_yuanyang_thinking` - 思考/沉思

**服装变体**
- 初中校服 (白衬衫+裙子)
- 高中校服 (不同款式)
- 便装-夏 (T恤+短裤/裙子)
- 便装-冬 (毛衣+外套)
- 运动装 (跑步场景)
- 正装 (重要场合)

**建议规格**
- 格式: PNG (透明背景)
- 分辨率: 1000×2000 或 1200×2400
- 文件大小: 每个 < 2MB
